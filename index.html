  • Verify sender addresses - Check the email domain carefully
  • Hover over links - See the actual URL before clicking
  • Use 2-factor authentication - Add extra security layer
  • Keep software updated - Install security patches
  • Use strong, unique passwords - Different for each account
  • Report suspicious emails - Help protect others
  • Contact companies directly - Use official phone numbers
  • Trust your instincts - If it feels wrong, it probably is

‚ùå Don'ts

  • Don't click suspicious links - Even if they look legitimate
  • Don't download unknown attachments - They may contain malware
  • Don't share personal information via email - Banks never ask this way
  • Don't use public WiFi for sensitive transactions - Use VPN if necessary
  • Don't reply to suspicious emails - It confirms your email is active
  • Don't trust caller ID - It can be spoofed
  • Don't panic under pressure - Scammers use urgency tactics
  • Don't reuse passwords - One breach affects all accounts
